Security Engine

Zero-tolerance anti-nuke protection

Zerian watches every audit log event in real-time. The moment someone starts mass-deleting channels, mass-banning members, or granting themselves admin — they're stopped instantly.

  • Mass channel/role delete & create detection
  • Mass ban & kick detection
  • Unauthorized permission escalation blocking
  • Webhook spam prevention
  • Emergency panic mode — locks server in one command
  • Quarantine: strips all roles + timeouts instantly
  • Role permission backup & restore snapshots
  • Full whitelist system for trusted admins
