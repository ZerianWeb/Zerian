Security Engine

Zero-tolerance anti-nuke — stops attacks before damage is done

Zerian watches every audit log event in real-time. Mass channel deletes, mass bans, permission grabs — the moment a threshold is crossed, the attacker is actioned instantly.

  • Mass channel/role delete & create detection
  • Mass ban & kick detection with configurable thresholds
  • Admin permission escalation blocked automatically
  • Webhook spam prevention
  • Emergency panic mode — locks server + strips dangerous perms in one command
  • Quarantine: strip all roles + timeout instantly, restore with one command
  • Role permission backup & restore snapshots
  • Trusted admin whitelist so your mods are never flagged

Anti-Link, Banned Words & Mentions
